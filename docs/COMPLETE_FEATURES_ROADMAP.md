```http
POST   /api/savingsgoals/{id}/deposit       - Add deposit
```

### **Recurring Bills Endpoints:**
```http
GET    /api/recurringbills                  - Get all bills
GET    /api/recurringbills/{id}             - Get specific bill
POST   /api/recurringbills                  - Create bill
PUT    /api/recurringbills/{id}             - Update bill
DELETE /api/recurringbills/{id}             - Delete bill
POST   /api/recurringbills/{id}/mark-paid   - Mark as paid
GET    /api/recurringbills/upcoming?days=30 - Get upcoming
GET    /api/recurringbills/summary          - Get summary stats
```

### **Loan Payments Endpoints:**
```http
GET    /api/loanpayments                    - Get all payments
GET    /api/loanpayments/{id}               - Get specific payment
GET    /api/loanpayments/by-loan/{loanId}   - Get loan payments
POST   /api/loanpayments                    - Create payment
PUT    /api/loanpayments/{id}               - Update payment
DELETE /api/loanpayments/{id}               - Delete payment
GET    /api/loanpayments/summary/{loanId}   - Get payment summary
```

### **Shopping Lists Endpoints:**
```http
GET    /api/shoppinglists                   - Get all lists

PUT    /api/shoppinglists/{id}              - Update list
DELETE /api/shoppinglists/{id}              - Delete list
POST   /api/shoppinglists/{id}/complete     - Complete list
GET    /api/shoppinglists/summary           - Get summary

GET    /api/shoppinglists/{listId}/items                     - Get items
POST   /api/shoppinglists/{listId}/items                     - Add item
PUT    /api/shoppinglists/{listId}/items/{itemId}            - Update item
DELETE /api/shoppinglists/{listId}/items/{itemId}            - Delete item
POST   /api/shoppinglists/{listId}/items/{itemId}/toggle     - Toggle checked
```
